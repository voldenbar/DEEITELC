# 项目简介

## 核心功能
- 健身知识推荐：采用协同过滤算法，为用户提供个性化的健身知识推荐。
- 智能AI集成：集成百度千帆大模型，提供智能化的健身建议和互动。

## 适用场景
- 健身爱好者：通过系统获取个性化的健身知识和建议。
- 健身教练：利用系统管理用户数据，提供定制化的健身计划。

## 技术特点
- 架构：采用B/S架构，MVC模式，Web框架设计。
- 开发环境：支持IDEA、JDK1.8、Mysql、Maven、Navcat等开发工具。
- 技术栈：基于Java、SpringBoot、Vue等技术栈构建。
